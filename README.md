# RAG-Powered ChatBot

## Introduction
RAG ChatBot implemented with a StreamLit dashboard in a docker container.

## Setup
Project can be run locally with the following commands. Recommended to create a fresh virtual environment.

```bash
# Install dependencies
pip3 install -r requirements.txt

# Run the dashboard
streamlit run src/app.py
```

Project also supports a docker container by building and running with the following commands.

```bash
# Build the docker image

# Run the image.
```
